# kiepski-quotes

> Random quotes from Swiat Wedlug Kiepskich

## Install

```shell
npm i kiepski-quotes
```

## Usage

```js
import kiepski from "kiepski-quotes";

kiepski.getAll(); // Panie, noc jest!
kiepski.getFerdek();
kiepski.getHalinka();
kiepski.getEdzio();
kiepski.getArnold();
kiepski.getHelena();
kiepski.getMarian();
```
